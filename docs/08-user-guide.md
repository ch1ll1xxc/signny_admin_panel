# Руководство пользователя АК АС «Сайнни»

## 1. Вход в систему

1. Откройте адрес системы в браузере (по умолчанию `http://localhost:5173`)
2. На странице входа выберите роль: **Администратор**, **Редактор** или **Куратор**
3. Нажмите соответствующую кнопку входа
4. Вы будете перенаправлены на дашборд

### Роли и права

| Роль | Возможности |
|------|------------|
| Администратор | Полный доступ: все разделы, публикация, удаление |
| Редактор | Создание/редактирование экспонатов, FAQ, отправка на согласование |
| Куратор | Согласование, возврат на доработку, комментарии |

## 2. Дашборд

Отображает сводку по системе:
- **Версий в работе** — общее количество версий контента
- **На согласовании** — версии, ожидающие решения куратора
- **Нужна доработка** — версии, возвращённые на исправление
- **Опубликовано** — опубликованные версии
- **Быстрые действия** — ссылки на основные разделы

## 3. Управление экспонатами

### Просмотр каталога
- Перейдите в раздел **Экспонаты** в сайдбаре
- Используйте фильтры: поиск по названию, фильтр по экспозиции, фильтр по статусу

### Создание экспоната
1. Нажмите **Создать экспонат**
2. Введите название
3. Нажмите **Создать**

### Редактирование карточки
1. Кликните на экспонат в таблице
2. Нажмите **Редактировать** в секции «Карточка экспоната»
3. Измените поля: название, описание, URL изображения
4. Нажмите **Сохранить**

### Работа с контентом версии
1. В секции «Контент версии» нажмите **Редактировать контент**
2. Заполните **Исходный текст (русский)**
3. При необходимости отредактируйте **Адаптированный текст (для РЖЯ)**
4. Нажмите **Сохранить контент**

### Предобработка текста для РЖЯ
1. Убедитесь, что исходный текст заполнен
2. В секции «Задачи» нажмите **Запустить предобработку**
3. Система упростит текст для представления на РЖЯ
4. Результат появится в поле «Адаптированный текст»
5. При необходимости отредактируйте результат вручную

### Генерация QR-кода
1. В секции «QR-код» нажмите **Сгенерировать QR**
2. QR-код отобразится на экране
3. Нажмите **Скачать PNG** для сохранения файла
4. Нажмите **Печать** для печати QR-кода

## 4. Согласование (модерация)

### Для редактора
1. На карточке экспоната нажмите **Отправить на согласование**
2. Версия перейдёт в статус «На согласовании»

### Для куратора
1. Перейдите в раздел **Модерация**
2. Просмотрите версии в списке «Ожидают модерации»
3. Введите комментарий в текстовое поле
4. Нажмите **Согласовать** или **На доработку**

## 5. Управление FAQ

1. Перейдите в раздел **FAQ**
2. Нажмите **Добавить FAQ** для создания нового вопроса
3. Выберите экспонат, заполните вопрос и ответ
4. Опционально добавьте URL видео и субтитры
5. Для публикации/снятия нажмите кнопку **Опубликовать**/**Скрыть**

## 6. Публикация

1. Перейдите в раздел **Публикация**
2. Нажмите **Запустить проверку** — система покажет количество версий по статусам
3. Нажмите **Опубликовать всё согласованное** для массовой публикации
4. Система автоматически синхронизирует данные с публичным контуром

### Привязка аватарных материалов
1. В секции «Аватарные материалы — экспонаты» нажмите **Привязать** у нужного экспоната
2. Укажите URL сцены с аватаром
3. Нажмите **Сохранить**
4. Аналогично для FAQ в секции «Аватарные материалы — FAQ»

## 7. Журнал аудита

1. Перейдите в раздел **Аудит**
2. Просматривайте все действия: время, пользователь, действие, объект, результат
3. Фильтруйте по пользователю и результату (успех/отклонено)

## 8. Аналитика

Раздел **Аналитика** отображает:
- Метрические карточки: количество версий, статусы, действия
- Последняя активность workflow

## 9. Выгрузки

1. Перейдите в раздел **Выгрузки**
2. Выберите тип выгрузки: **Статусы версий**, **Журнал аудита**, **FAQ**
3. Нажмите **Скачать CSV**
4. Файл будет скачан в формате CSV (UTF-8, совместим с Excel)

## 10. Медиатека

1. Перейдите в раздел **Медиатека**
2. Просматривайте загруженные файлы (видео, субтитры)
3. Используйте поиск для фильтрации
4. Кликните на файл для просмотра деталей

## 11. Залы

1. Перейдите в раздел **Залы**
2. Нажмите **Добавить зал** для создания нового
3. Заполните название и код
4. Данные сохраняются автоматически
