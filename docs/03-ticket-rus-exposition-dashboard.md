# Ticket: Русификация + Экспозиция/Экспонат + Дашборды

## Goal
Привести демо-админку к предметному языку музея: русскоязычный интерфейс, явное разделение сущностей "Экспозиция" и "Экспонат", а также выразительные дашборды с подготовкой к интеграции Grafana.

## Acceptance Criteria
- Интерфейс ключевых экранов и навигации отображается на русском языке.
- В модели данных есть `Экспозиция` и связанные `Экспонаты`.
- В таблицах/карточках видно, к какой экспозиции относится экспонат.
- Dashboard и Analytics показывают визуальные метрики (статусы, динамику, топы, operational панели).
- Есть явные "grafana-ready" метрики/панели для дальнейшего подключения.

## Constraints
- Реализация в текущем Vue mock-контуре, без реального backend.
- Минимальные, но рабочие изменения без ломки существующих маршрутов.
